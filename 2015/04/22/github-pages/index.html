Starting github pages as blog

1. Install git

安装git
关联github账号
在github里添加本机Key

2. Install nodejs

一键安装

3. Install hexo

右键->Git Bash
npm install -g hexo
npm install hexo—save //类似这个会有提示
重启
hexo init
修改_config.yml //设置远程repo为github

deploy:
type: git
repo: git@github.com:felacebo/felacebo.github.io.git
branch: master

hexo new “first post” //会在source_posts生成”first-post.md”
hexo g //generate
hexo s //deploy to local server
hexo d //deploy to remote server

Enjoy it ~
